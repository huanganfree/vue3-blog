## 技术
- 前端：vue3 + vant + vue-router4
- 后端：node + express + mysql + sequelize + pm2

## 快速开始
### 前端服务启动
```
npm run serve
```
### 后端服务启动
进入到server目录下

```
npm run start:server
```
